== Description ==

RB Portfolio One is a sophisticated WordPress theme crafted specifically for freelancers, designers, developers, and creative professionals who want to build a stunning online presence. With its clean aesthetics and robust feature set, this theme provides everything you need to showcase your portfolio while maintaining excellent performance and user experience.

**Key Features:**
- **Responsive Design** – Flawless display across all devices and screen sizes
- **Portfolio-Focused** – Showcase your projects with style and professionalism
- **Blog-Ready** – Share insights and updates with a beautifully formatted blog
- **Customizable** – Easy customization via WordPress Customizer
- **Custom Logo Support** – Upload your personal or business branding
- **Custom Menus** – Create intuitive navigation structures
- **Featured Images** – Support for eye-catching post and page thumbnails
- **Right Sidebar** – Optimized layout with widget-ready sidebar
- **One-Column Layout** – Clean, distraction-free content presentation
- **Lightweight & Fast** – Optimized code for quick loading times

Whether you're a photographer, web designer, artist, writer, or any type of creative professional, RB Portfolio One provides the perfect foundation for your professional website.

== Installation ==

**From your WordPress Dashboard:**
1. Navigate to **Appearance → Themes**
2. Click the **Add New** button
3. In the search field, type **"RB Portfolio One"**
4. Once located, click the **Install** button
5. After installation, click **Activate** to start using the theme
6. Go to **Appearance → Customize** to personalize your theme settings

**Manual Installation via FTP:**
1. Download the theme zip file to your computer
2. Extract the contents and upload the `rb-portfolio-one` folder to `/wp-content/themes/`
3. Go to **Appearance → Themes** in your WordPress admin
4. Find RB Portfolio One and click **Activate**
5. Navigate to **Appearance → Customize** to configure your theme

== Frequently Asked Questions ==

**Do I need any plugins to use this theme?**
The theme works perfectly out of the box. For enhanced functionality, you may optionally install the RB Theme Helper plugin (recommended) which provides additional Kirki Customizer features.

**Is this theme translation-ready?**
Yes, RB Portfolio One follows WordPress internationalization best practices and is fully translation-ready.

**Can I use this for a business website?**
Absolutely! While optimized for portfolios, the clean design works great for business websites, personal blogs, and professional landing pages.

**Where can I get support?**
For support, please visit the WordPress.org support forums or contact the theme author through their profile at https://bashir-rased.com/
